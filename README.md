Voice Biometrics
======================

# How to install
	open command prompt and navigate to the project directory
	cd group-voice-biometrics/
	run	npm install
	run	bower install
	run	grunt serve - if it fails runt grunt serve --force
	http://localhost:9000/

# Tech stack
Node.js, AngularJS, Express 4, Bootstrap, Jade, Yeoman, Grunt, Bower
